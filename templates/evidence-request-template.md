# Evidence Request Template

## Control Name
Example: IA-2 Identification and Authentication

## Evidence Requested
- MFA configuration screenshot
- User authentication policy
- Access control list
- Recent access review report
- System-generated logs

## Assessment Method
- Examine
- Interview
- Test

## Evidence Quality Criteria
Evidence should be current, complete, timestamped where possible, and directly tied to the control requirement.

## Notes
Policy alone is not sufficient when technical enforcement is required.
